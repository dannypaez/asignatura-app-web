Guía Académica Extendida de Clases Utilitarias en Tailwind CSS

Categoría Clases Descripción académica
Espaciado: Margin m-0 a m-96, mx-4, my-8 Define espacio externo (margen) alrededor de un elemento.
Espaciado: Padding p-0 a p-96, px-4, py-8 Define espacio interno (relleno) dentro de un elemento.
Espaciado: Gap gap-0 a gap-96 Define espacio entre elementos hijos en Flexbox o Grid.
Tamaños: Width w-0 a w-96, w-full Define el ancho de un elemento.
Tamaños: Height h-0 a h-96, h-screen Define la altura de un elemento.
Tamaños: Min/Max min-w-0 a min-w-full, max-h-0 a max-h-full Establece límites mínimos y máximos para ancho y alto.
Tipografía text-xs a text-9xl, font-thin a font-black, italic, underline, uppercase, tracking-wide Modifica tamaño, peso, estilo, decoración y espaciado de texto.
Colores text-red-700, bg-blue-500, border-green-300, bg-opacity-0 a bg-opacity-100 Asigna colores y niveles de opacidad a fondo, texto o bordes.
Bordes border, border-2 a border-8, border-t, border-none, rounded, rounded-lg, rounded-full Define grosor, presencia y redondeado de los bordes.
Layout block, inline-block, inline, hidden, container, box-border, box-content Controla el comportamiento y tamaño del contenedor o elemento.
Flexbox flex, flex-row, flex-col, justify-center, items-center, flex-wrap Permite diseños flexibles y alineación de elementos.
Grid grid, grid-cols-1 a grid-cols-12, gap-x-4, gap-y-2 Crea sistemas de diseño basados en cuadrículas.
Posicionamiento relative, absolute, fixed, sticky, top-0 a top-full, inset-0 Define cómo se ubica un elemento respecto a su contenedor.
Sombras y Opacidad shadow-sm a shadow-2xl, opacity-0 a opacity-100 Agrega profundidad visual o controla transparencia.
Transiciones transition, duration-75 a duration-1000, ease-linear, ease-in-out Aplica efectos de suavizado al cambiar estilos.
Animaciones animate-bounce, animate-spin, animate-ping, animate-pulse Aplica animaciones predefinidas de movimiento o transformación.
Estado hover:bg-blue-500, focus:outline-none, active:bg-red-500, disabled:opacity-50 Aplica estilos cuando el usuario interactúa con un elemento.
Visibilidad y Display hidden, block, inline, visible, invisible Muestra u oculta elementos o cambia su tipo de display.
Interactividad cursor-pointer, pointer-events-none, select-none Controla interacción de puntero y selección de texto.
Orden y Overflow z-0 a z-50, overflow-hidden, overflow-auto, overflow-scroll Define el orden de apilamiento y cómo manejar desbordes de contenido.
Misceláneas aspect-ratio, isolate, mix-blend-mode, backdrop-blur Clases avanzadas para controlar relaciones de aspecto, capas y efectos visuales.
