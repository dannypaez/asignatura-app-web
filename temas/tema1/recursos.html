Principales Etiquetas HTML y Propiedades CSS

📄 Tabla de Etiquetas HTML Principales

Etiqueta Descripción
<!DOCTYPE html> Define el tipo de documento como HTML5.
<html> Elemento raíz de una página web.
<head> Contiene metadatos e información del documento.
<title> Título de la página que aparece en la pestaña del navegador.
<meta> Información adicional como codificación o descripción SEO.
<link> Enlaza archivos externos (por ejemplo, CSS).
<script> Agrega o enlaza código JavaScript.
<body> Contiene todo el contenido visible de la página.
<h1> a <h6> Encabezados de diferentes niveles de importancia.
<p> Define un párrafo de texto.
<a> Define un enlace o hipervínculo.
<img> Inserta una imagen.
<div> Contenedor genérico en bloque.
<span> Contenedor genérico en línea.
<section> Agrupa contenido relacionado en una página.
<article> Contenido independiente o autosuficiente.
<header> Encabezado de una sección o página.
<footer> Pie de página o final de una sección.
<nav> Contiene enlaces de navegación.
<main> Contenido principal de la página.
<ul> Lista desordenada.
<ol> Lista ordenada.
<li> Elemento de una lista.
<form> Contenedor de un formulario.
<input> Campo de entrada de datos.
<textarea> Campo de texto de varias líneas.
<button> Botón interactivo.
<select> Menú desplegable.
<option> Opción dentro de un <select>.
<label> Etiqueta para un elemento de formulario.
<br> Salto de línea.
<hr> Línea horizontal divisora.
<strong> Texto enfatizado (semánticamente fuerte).
<em> Texto enfatizado (énfasis en voz).
<blockquote> Cita en bloque.

🎨 Tabla de Propiedades CSS Principales

Propiedad Descripción
margin Espacio exterior alrededor del elemento.
padding Espacio interior entre el contenido y el borde.
border Borde alrededor del elemento.
width Ancho del elemento.
height Alto del elemento.
box-sizing Cómo se calculan el ancho y el alto (border-box o content-box).
background-color Color de fondo del elemento.
color Color del texto.
font-family Familia tipográfica del texto.
font-size Tamaño de la fuente.
font-weight Grosor del texto.
text-align Alineación del texto (izquierda, centro, derecha).
text-decoration Decoración del texto (por ejemplo, subrayado).
line-height Altura de línea (espaciado vertical entre líneas de texto).
letter-spacing Espaciado entre letras.
display Cómo se muestra un elemento (block, inline, flex, etc.).
position Posicionamiento (static, relative, absolute, fixed, sticky).
top, right, bottom, left Coordenadas para posicionar elementos.
z-index Orden de apilamiento (qué elemento se ve encima).
overflow Manejo del contenido que desborda el contenedor.
border-radius Bordes redondeados.
box-shadow Sombra para los elementos.
transition Efectos de transición suave en cambios de estado.
cursor Tipo de cursor del mouse al pasar sobre el elemento.
visibility Visibilidad del elemento (visible, hidden).
opacity Transparencia del elemento.
flex-direction Dirección de los elementos en un contenedor flex (row, column).
justify-content Alineación horizontal de los elementos en flexbox.
align-items Alineación vertical de los elementos en flexbox.
gap Espacio entre elementos hijos en flexbox o grid.
max-width Ancho máximo permitido.
min-width Ancho mínimo permitido.
@media Media Queries: aplica estilos según el tamaño de pantalla o dispositivo.
